#### 支持功能

1. 单聊和群聊，新增消息管理器

2. 支持发送表情、图片、语音、视频和文件消息

3. 单聊支持消息已读未读的状态显示

4. 支持设置新消息声音提醒，浏览器通知

5. 支持部分Lemon-imui内功能设置

6. 支持文件、图片和绝大部分媒体文件在线预览

7. 群聊创建、删除和群成员管理、群公告、群禁言等

8. 可以置顶联系人，所有联系人可以设置消息免打扰

#### 安装教程
1.  克隆代码到本地： 
``` 
git clone https://gitee.com/raingad/im-chat-front.git
```
2.  进入项目目录，执行： 
```
npm intsall
```
3.  开发调试
```
npm run serve
```

4.  构建
```
npm run build
```
